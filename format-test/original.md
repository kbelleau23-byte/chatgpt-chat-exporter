# Newline character example

**Date:** 2026-07-06
**Source:** chatgpt.com

---

### **You**

What is \n ? Show me a 5-line example. Make no mistakes.

---

### **ChatGPT**

`\\n` is the **newline character** (also called a **line feed**, LF).

When a program outputs `\\n`, it moves the cursor to the beginning of the **next line**.

Example (5 lines of output):


```
Line 1Line 2Line 3Line 4Line 5
```

The string that produces that output is:


```
"Line 1\nLine 2\nLine 3\nLine 4\nLine 5"
```

There are exactly **4** `\\n` characters in that string, because each `\\n` separates one line from the next, producing **5** lines of output.
